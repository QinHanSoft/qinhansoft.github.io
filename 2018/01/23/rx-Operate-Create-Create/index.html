Create

Jan 23, 2018

使用一个函数从头开始创建一个Observable。

你可以使用create操作符从头创建一个Observable，给这个操作符传递一个接受观察者作为参数的函数，编写这个函数让它的行为表现为一个Observable——恰当的调用观察者的onNext，onError和onCompleted方法。

一个形式正确的有限Observable必须尝试调用观察者的onCompleted正好一次或者它的onError正好一次，而且此后不能再调用观察者的其他任何方法。

RxJava将这个操作符实现为create方法。

建议在传递给create方法的函数中检查观察者的isUnsubscribed状态，一遍在没有观察者的时候，让Observable停止发射数据或者做昂贵的运算。
